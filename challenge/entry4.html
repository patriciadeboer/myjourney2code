Put together a basic tic tac toe game to play vs the computer.

Today's Project: Tic Tac Toe

The idea behind this project is a user can select a box to put a X and then the computer will respond with a O. As of right now, it doesn't stop when you hit 3 in a row

The Steps

  • Create basic css grid for the tic tac toe box
  • Create a JS file that slowly goes through each of the following:
    1. First have it so that once a box is selected, an X appears from the user
    2. Trigger a computer turn function that randomly selects a box and inserts a O
    3. Next start implementing checks.
      • If the user selects a filled box, alert user.
      • If computer selects filled box, make the function choose another
    4. Things still want to add:
      • Right now, the computer is randomly selecting a box, ideally it should be trying to block any multiple boxes in a row, but alas, I will save that for a future date :)
      • Choice to play with friend vs play with computer
      • Adding highlight of winning row once 3 in a row is in play

The Lessons

What I learned along the way:

  • make sure you have the right js file linked :)
  • New functionalities used:
    • using 'this' in js to identify the current selected box
    • using option to append within the box via javascript

And that's a wrap....
