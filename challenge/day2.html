Today's Project: Follow the Mouse

The idea behind this project is for a circle to follow the mouses movement with a slight delay.

The Steps

These are the steps I took for my project:

  1. Signed into my GitHub Account (reset my password because let's be honest I didn't remember that...)
  2. On my computer under my user folder, I created a new folder called '100_days_of_Code'
  3. Next I opened 'Terminal'. I usually do this by hitting Command + 'space bar' then typing in 'Terminal'
  4. I went to said folder by typing 'cd 100_days_of_Code'
  5. Next I initialized the git process by typing 'git init'
  6. I got a message saying Initialized empty Git repository. From there I typed 'git status' just to check in
  7. I had no files in my folder, but then decided to add a readme.md file. I copied one from another project from before, updated it, and next up was adding it to git
  8. Next used git add to include in what I wanted to commit. I like to do all files (at least at this point in the project) so simply typed git add -A
  9. Next I committed the file(s) and added a comment of what the update was for, in this case adding the readme.md file so entered git commit -m 'Adding a readme.md file'
  10. After getting confirmation of the commit, I was ready to push to the repository

And that's a wrap....
