Today's Project: Yahtzeeee

The idea behind this project is to be play Yahtzee. Given it's the holidays and the amount of features to be built in, this will definitely be a multi-day project. Happy holidays!

The Steps

Step 1: Create the dice face

  • Created 5 equal faces for the Yahtzee dice
  • Started with Numbers for now, maybe in future upgrade to actual dots on the dice. Started with dots but got stuck for now
  • Center and align the dice

Step 2: Create random number values for the dice and hold functionality

  • Create function that randomly selects a number between 1-6 and places them on the dice face
  • Make the dice faces clickable
  • If the face is clicked, change color of face to indicate it's selected. But also allow user to change mind and unclick face again
  • If a dice is 'held', ensure it doesn't change numbers (skip it from random number generation)

Create "Roll" button and limit the number of rolls

  • Give the user 3 rolls total per turn
  • Once the user has 0 rolls left, don't allow to roll again

Step 4: Create score table and calculate score

  • This has been the most fun part ..ok the most challenging so in some ways more fun, and some ways not
  • Had some issue with small Straight for some reason couldn't get my head around it. Not convinced I picked the easiest route, but that's ok. It works.
  • Output various score options for each dice combinations
  • Need to upgrade it still to allow storing of score

Step 5: Future:

  • Create a new game button
  • Create rues button (use alert?)
  • Make upgrades
    • add dots on the faces instead of numbers
    • Have the dice "roll" via animation

The Lessons

  • New functionalities used:
