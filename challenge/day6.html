Today's Project: Hangman

The idea is well to recreate the classic game of hangman, where the user will input letters to guess the current word. The user must guess within 6 guesses before the stick figure gets hung

The Steps

These are the steps I took for my project:

  1. Created a very basic HTML file
    • Added a Submit button that triggers a word to be selected when clicked
    • Inserted word placeholder to show word length via blank indicators such as _ _ _ _ _
    • Created an input field for user to enter guesses along with a submit button
    • Note: Both submit buttons trigger javascript functions
  2. Wrote a corresponding js file to store functions
  3. Functions include:
    • triggerWord()- which randomly selects a word from a list of 100 and returns a string of _ to indicate word length
    • hangman()- which takes the user input, and determines if it is part of the word or not. If it is, it replaces the blank _ with the letter in the correct spot, otherwise, returns the incorrect letter along with incorrect count.
    • Note: If the user guesses the correct answer, returns a greeting + image. If runs out of guesses, returns try again next time!
  4. Had to add some checks in such as if the user submits blanks, invalid values (non-alpha), more than one letter, repeats a letter. Also I added image of hangman figure to replicate the true hangman experience

The Lessons

  • Slowly getting better at using html and javascript together to make for a more interactive experience
  • How to share alerts and errors if the input isn't valid. Probably here as well could minimize the amount of code used to implement checks, but again, saving that for another day
  • How to pass variables between functions, and using global variables in order to ensure proper tracking of values as functions are ran multiple times
